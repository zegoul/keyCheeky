Welcome to Keycheeky

KeyCheeky is a lightweight, client-side demo credential manager built to show UI and flows. Use the Login or Sign Up pages to continue. After logging in you'll reach a dashboard where you can store simple credential entries (stored in your browser only).

Get Started  →

Demo Storage

Entries are persisted only in your browser using localStorage. This is a demo flow — do not store real secrets here.

Separate Pages

Login, Sign Up and Dashboard are now separate pages so navigation matches a real app structure.

Privacy-first

No external servers are contacted by this demo. All data stays on your machine unless you export it.

How to use (quick)

  1. Create an account via Sign Up (or use demo user on Login).
  2. Login — you'll be redirected to the Dashboard.
  3. From Dashboard use "Add password" or "See list" to manage entries.
