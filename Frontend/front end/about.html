About Keycheeky

Keycheeky is a minimal, client-only credential manager built for demonstration and design testing. It demonstrates common flows: sign up, login, a per-user dashboard, and CRUD-style password entries. The project intentionally keeps everything on the client (using localStorage) to be easy to inspect and modify while you build a backend.

Why a demo?

This demo helps you prototype UI and flows without committing to a backend. Once the UI is locked, we can wire it to a secure backend (for example an encrypted SQLite store behind an authenticated API).

Next steps

  • Export / Import entries to a file (JSON) so data can be moved between devices.
  • Replace localStorage with a backend API and add proper authentication.
  • Introduce encryption at rest (server-side or via WebCrypto) and a secrets vault.
